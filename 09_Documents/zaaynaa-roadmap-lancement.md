# 🚀 ZAAYNAA — ROADMAP DE LANCEMENT
## Plan complet par phases · 2 partenaires · Objectif 10-12 semaines

---

## 👥 RÉPARTITION DES RÔLES

```
TOI (CEO + CRÉATIF)
→ Designs produits, identité, contenu, site, marque
→ Direction artistique, storytelling

TON POTE (COO + OPÉRATIONS/FINANCE)
→ Sourcing, négociation fabricants, budget
→ Logistique, gestion commandes, comptes
```

---

## 🗓️ VUE D'ENSEMBLE DES PHASES

```
PHASE 0 — Fondations (continu, en parallèle)
PHASE 1 — Designs produits ✅ déjà très avancé
PHASE 2 — Sourcing & négociation fabricants
PHASE 3 — Validation samples
PHASE 4 — Contenu réel (photos/vidéos)
PHASE 5 — Site finalisé + boutique
PHASE 6 — Lancement (ads + réseaux)

DURÉE TOTALE VISÉE : 10-12 semaines
```

⚠️ **RÈGLE ABSOLUE : examens d'abord (23 jours).**
Les 3 premières semaines = rythme réduit sur Zaaynaa.
Après examens = on pousse fort.

---

# 🧱 PHASE 0 — FONDATIONS (Semaine 1, puis continu)

**Objectif :** poser le cadre pour ne pas s'effondrer après.

```
👥 ENSEMBLE :
☐ Valider le cadre partenariat 50/50 (en cours)
☐ Réunion de lancement (rôles, règles, deadlines)
☐ Créer un dossier Google Drive partagé "ZAAYNAA"
☐ Créer un tableau de suivi partagé (Google Sheet)
☐ Définir le budget précis (qui met quoi, quand)

👤 TOI :
☐ Centraliser tous les documents (brand book, prompts, designs)
☐ Réunir tes images produits dans le Drive

👤 TON POTE :
☐ Créer compte Alibaba
☐ Préparer un tableur de suivi fabricants
☐ Setup compte bancaire / moyen de paiement international
```

⏱️ **Durée : continu, ~3-4h chacun la 1ère semaine**

---

# 🎨 PHASE 1 — DESIGNS PRODUITS (Semaines 1-3)

**Objectif :** finaliser la collection de lancement.
**Statut : ✅ tu es déjà à 70% !**

```
👤 TOI :
☐ Finaliser SABR V2 + HILAL V2 (corrections en cours)
☐ Générer HOB ÉMERAUDE + NOUR SIGNATURE (hero products)
☐ Choisir les 3-5 produits du LANCEMENT
   (ne lance pas 20 produits, commence focus)
☐ Préparer les specs techniques de chaque produit
   (matière, dimensions, gravure)

⚠️ Recommandation : LANCE avec 3-5 produits max
   → 2 Héritage (sans pierre) + 1-2 Nour (avec pierre)
   → Tu ajoutes le reste après
```

⏱️ **Durée : 1-3 semaines (par petites sessions)**
**Priorité basse pendant les examens.**

---

# 🏭 PHASE 2 — SOURCING & NÉGOCIATION (Semaines 3-5)

**Objectif :** trouver le fabricant et commander les samples.
**Pilote : TON POTE (avec ton input sur la qualité visée)**

```
👤 TON POTE (principal) :
☐ Rechercher fabricants Alibaba (argent 925 + vermeil)
☐ Contacter 8-10 fabricants (email template fourni)
☐ Scorer les réponses (grille /100)
☐ Sélectionner 3 finalistes
☐ Négocier prix, MOQ, délais
☐ Commander les SAMPLES (3-5 pièces)

👤 TOI (support) :
☐ Fournir les designs + specs aux fabricants
☐ Valider que le fabricant comprend le style
☐ (optionnel) Faire les fichiers CAD via Fiverr si besoin

📌 Ton ami en Chine = atout ici
   → Il peut recevoir/vérifier les samples avant envoi
```

⏱️ **Durée : 2-3 semaines (incluant délais de réponse)**
💰 **Budget samples : 250-350€**

---

# 🔍 PHASE 3 — VALIDATION SAMPLES (Semaines 5-7)

**Objectif :** vérifier la qualité réelle, choisir le fabricant final.

```
👥 ENSEMBLE :
☐ Recevoir les samples
☐ Évaluer chaque sample (checklist qualité)
☐ Comparer les fabricants
☐ Choisir LE fabricant final

👤 TOI :
☐ Comparer samples vs tes images (qualité visuelle)
☐ Valider l'esthétique, les finitions, la gravure

👤 TON POTE :
☐ Évaluer le rapport qualité/prix
☐ Vérifier solidité, conformité specs
☐ Si OK → négocier la commande de production
☐ Si KO → relancer un autre fabricant
```

⏱️ **Durée : 1-2 semaines (incluant livraison)**

⚠️ **NE SAUTE JAMAIS cette phase.** Un mauvais sample 
non détecté = 50 pièces invendables.

---

# 📅 TIMELINE VISUELLE (12 semaines)

```
SEM 1-3  : 🎓 EXAMENS (rythme réduit)
           Phase 0 (fondations) + Phase 1 (designs, doucement)

SEM 4    : 🎉 Post-examens — on pousse
           Phase 1 finalisée + Phase 2 démarre (Alibaba)

SEM 5-6  : Phase 2 (négociation) + Phase 3 (samples commandés)
           Phase 5 (site) démarre en parallèle (apprentissage)

SEM 7    : Phase 3 (validation samples)
           Phase 4 (contenu) démarre

SEM 8-9  : Phase 4 (contenu) + Phase 5 (site finalisé)
           Commande production lancée

SEM 10   : Réception production + contrôle qualité
           Site finalisé, derniers tests

SEM 11   : Préparation lancement (contenu prêt, ads prêtes)

SEM 12   : 🚀 LANCEMENT ZAAYNAA
```

# ✅ CE QUE J'AI CORRIGÉ DANS TON PLAN

```
TON ORDRE INITIAL :
0-Site → 1-Designs → 2-Vidéos ads → 3-Fabricants → 5-Ads

❌ Problème : vidéos/ads AVANT d'avoir les produits réels
❌ Problème : fabricant trop tard (phase 3)
❌ Problème : site "fini" avant d'avoir produits/prix

MON ORDRE CORRIGÉ :
0-Fondations → 1-Designs → 2-Fabricants → 3-Samples 
→ 4-Contenu réel → 5-Site fini → 6-Lancement

✅ Le fabricant vient TÔT (tu as besoin des samples)
✅ Le contenu réel vient APRÈS les samples
✅ Le site se REMPLIT à la fin (mais se construit tôt)
✅ Les ads en DERNIER (quand tout est prêt)
```

# 🎯 RÈGLES D'OR

```
1. Examens d'abord (23 jours). Non négociable.
2. Lance avec 3-5 produits, pas 20.
3. JAMAIS de production sans valider un sample.
4. Le contenu réel a besoin des samples → patience.
5. Réunion hebdo avec ton pote (dimanche soir).
6. Si une phase prend du retard, on décale, on ne bâcle pas.
7. Garde 20% du budget en réserve (imprévus).
```
