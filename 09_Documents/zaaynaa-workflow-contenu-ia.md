# 🎥 WORKFLOW 1 — VIDÉO PRODUIT
## (Le bijou qui tourne, zoom, lumière cinématique)

### Objectif
Une vidéo de 5-10 sec montrant un bijou Zaaynaa
de façon cinématique pour Instagram Reels / TikTok / pub.

### ÉTAPE 1 — Point de départ : ta photo DALL-E
```
Tu as déjà tes images produit (HOB, HILAL, NOUR...).
On va les ANIMER en vidéo.
```

### ÉTAPE 2 — Anime l'image (image-to-video)
```
OUTIL : Kling 3.0 (ou Runway / Pika)

Tu uploades ta photo produit (le hero shot)
Tu colles un prompt vidéo :

PROMPT VIDÉO EXEMPLE (HOB) :
"Cinematic slow rotation of a delicate gold ring 
on ivory marble. Soft morning light slowly moving 
across the gold surface. Camera slowly zooms in 
toward the arabic engraving. Luxury jewelry 
advertisement style. Smooth, elegant, 8 seconds."

→ Tu obtiens une vidéo de 5-8 sec
```

### ÉTAPE 3 — Voix off (optionnel)
```
OUTIL : ElevenLabs

Texte à faire lire (je te le rédige) :
"HOB. L'amour gravé dans l'or. 
 Une déclaration discrète, mais éternelle. 
 ZAAYNAA."

→ Choisis une voix féminine douce française
→ Télécharge le fichier audio
```

### ÉTAPE 4 — Montage final
```
OUTIL : CapCut (gratuit)

☐ Importe la vidéo (Kling)
☐ Importe la voix off (ElevenLabs)
☐ Ajoute une musique douce (bibliothèque CapCut)
☐ Ajoute le texte à l'écran :
   "HOB حب" puis "Elegance by Nature"
☐ Ajoute le logo ZAAYNAA en fin
☐ Sous-titres auto (si voix off)
☐ Export 1080×1920 (format vertical)
```

### ÉTAPE 5 — Publication
```
☐ Instagram Reels
☐ TikTok
☐ Story
→ Caption fournie par Claude
```

# 🎤 WORKFLOW 2 — VIDÉO UGC
## (Une femme qui parle de Zaaynaa)

### ⚠️ Important : ta copine ne peut pas être filmée
```
Donc on utilise des ACTEURS IA.
C'est exactement pour ça que Creatify/Arcads existent.
```

### ÉTAPE 1 — Le script (Claude le rédige)
```
Je te rédige un script UGC de 20-30 sec.

EXEMPLE DE SCRIPT UGC :
"Honnêtement, je cherchais une bague qui me ressemble.
 Pas un truc basique qu'on voit partout.
 Et je suis tombée sur ZAAYNAA.
 Regardez ce détail... [montre la bague]
 Il y a une calligraphie arabe gravée. 'Amour'.
 Et à l'intérieur, le nom de la marque.
 C'est discret, mais quand on regarde de près... wow.
 Franchement, pour le prix, la qualité est folle.
 Je vous mets le lien."
```

### ÉTAPE 2 — Génère la vidéo UGC
```
OUTIL : Creatify (ou Arcads)

☐ Choisis un acteur IA (femme, 25-35 ans, style européen/arabe)
☐ Colle le script
☐ L'IA génère la femme qui parle
☐ Tu ajoutes des plans de ta bague (B-roll)
   entre les moments où elle parle
```

### ÉTAPE 3 — Insère tes plans produit
```
OUTIL : CapCut

☐ Vidéo de l'actrice qui parle (Creatify)
☐ Coupe et insère tes vidéos produit (Workflow 1)
   aux moments clés ("regardez ce détail...")
☐ Musique de fond douce
☐ Sous-titres dynamiques (style TikTok)
☐ Export vertical
```

### ÉTAPE 4 — Publication
```
☐ TikTok (UGC marche très bien)
☐ Instagram Reels
☐ Pub Meta/TikTok (plus tard)
```
