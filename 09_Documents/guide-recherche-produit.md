# 📚 INTRODUCTION — Pourquoi ce guide est différent

La majorité des "guides recherche produit" te disent :
- *"Va sur AliExpress trier par best-sellers"*  ❌
- *"Cherche des produits avec WOW factor"*  ❌
- *"Trouve un winner dans Minea"*  ❌

**C'est de la merde.** Voici pourquoi :

- AliExpress best-sellers = saturé sur tous les marchés
- "WOW factor" est subjectif et pas mesurable
- Minea/AdSpy = tu copies des produits déjà vendus = tu arrives en retard

**La VRAIE méthode pro** que tu vas apprendre ici utilise **3 angles croisés** :

```
1. DEMANDE PROUVÉE (pubs actives en France depuis +30 jours)
2. ALIGNEMENT PERSONNEL (ton univers gaming)
3. FAISABILITÉ TECHNIQUE (sourcing EU + marges réelles)
```

Si un produit valide les 3 → c'est un vrai winner potentiel.

# ⚠️ LES 5 PIÈGES À ÉVITER

## Piège #1 : Le "winner" trop évident

Si TOUS les YouTubeurs en parlent ce mois-ci → trop tard. Cherche les produits qu'ils **ne** mentionnent **pas** encore.

## Piège #2 : Le produit "que toi tu veux vendre"

Le marché s'en fout de ce que TU veux vendre. Le marché veut ce qu'IL veut acheter. Suis les données, pas tes envies.

## Piège #3 : La saturation invisible

Un produit peut sembler peu saturé en surface mais être ultra-saturé en pubs. Vérifie TOUJOURS la concurrence Google + Facebook Ad Library.

## Piège #4 : Les fausses marges

Marge brute ≠ profit. Calcule :
```
Profit = Prix vente - (Coût achat + Livraison + Pub + Frais Shopify + Frais paiement)
```

Souvent le profit réel est 30-40% du prix vente, pas 60%.

## Piège #5 : L'analyse paralysie

Tu peux passer 3 semaines en recherche produit. **2 jours suffisent.** Au-delà, tu procrastines.
