# 📁 ZAAYNAA — Espace partagé

Dossier de travail partagé entre les deux associés.
Règle simple : chaque fichier va dans son dossier numéroté. Pas de fichiers en vrac à la racine.

## Structure
- **01_Brand/** — Identité de marque : brand book, logo, palette, storytelling FR/EN
- **02_Designs/** — Tous les designs produits (Bagues, Colliers, Boucles, Bracelets) + _Rejetes
- **03_Specs_Techniques/** — Une fiche technique PDF par produit
- **04_Fabricants/** — Scoring fabricants, emails, devis
- **05_Samples/** — Photos + évaluation qualité des échantillons reçus
- **06_Contenu/** — Instagram, Vidéos, Ads
- **07_Site/** — Code / lien GitHub + assets HD
- **08_Finance/** — Budget, factures, commandes
- **09_Documents/** — Tous les docs de référence (stratégie, guides, roadmap)

## Convention de nommage
`PRODUIT_VERSION_AAAA-MM-JJ.ext` → ex : `HOB_V2_2026-06-28.png`
